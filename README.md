# 校本中文科教師手冊 GitHub Pages 發佈版

這個資料夾只包含可用作 GitHub Pages 發佈的靜態網站檔案。

## 發佈方式

1. 在 GitHub 建立一個 repository。
2. 只把本資料夾 `github-pages/` 內的內容放入 repository。
3. 到 GitHub repository 的 Settings > Pages。
4. Source 選擇 `Deploy from a branch`。
5. Branch 選擇 `main`，Folder 選擇 `/ (root)`。
6. 儲存後等待 GitHub Pages 產生網址。

## 重要提醒

- 不要把上一層的 `COT 觀課後報告/`、`data/`、`tools/` 推上 GitHub。
- 原始 PDF、抽取文字及處理資料屬校內工作資料，應留在本機或學校核准的位置。
- 如果 GitHub Pages 網站會公開，發布前必須確認手冊內容已適合公開瀏覽。
